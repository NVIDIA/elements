# Themes (beta)

Themes are built leveraging CSS Custom Properties.

```css
@import '@elements/elements/index.css';
```

Themes are applied by setting the `mlv-theme` attribute on the host `html` element.

```html
<html lang="en" mlv-theme="dark">
```

Currently there are the following themes:
- Light (default)
- Dark
- High Contrast

Themes can be applied dynamically based on user preferences.

```javascript
if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
  body.setAttribute('mlv-theme', 'dark');
}

if (window.matchMedia('(-ms-high-contrast: active)')).matches || window.matchMedia('(forced-colors: active)')).matches) {
  document.body.setAttribute('mlv-theme', 'high-contrast');
}
```
