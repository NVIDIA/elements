To use Elements in [Remix](https://remix.run/) follow the [instalation](?path=/docs/integrations-react--docs) steps. Once complete
elements can be imported and used within React jsx and tsx files.

A experimental [@lit-labs/ssr-react](https://github.com/lit/lit/tree/main/packages/labs/ssr-react) package exists for rendering
elements with Remix SSR.

```javascript
// app/root.tsx
import '@lit-labs/ssr-react/enable-lit-ssr.js';

// the rest of your app
```

```json
// tsconfig.json
{
  "compilerOptions": {
    "jsxImportSource": "@lit-labs/ssr-react",
  }
}
```
