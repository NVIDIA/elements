# @nvidia-elements/media

Media playback UI components for Elements.

## Installation

```bash
npm install @nvidia-elements/media
```

## Usage

```typescript
import '@nvidia-elements/media';
```

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build project
pnpm run build

# Run tests
pnpm run test

# Run linter
pnpm run lint
```
