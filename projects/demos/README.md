# Demo Applications

This directory contains a suite of standardized demo apps for kickstarting and using Elements in a variety of Web technologies.

- `/typescript` - HTML + CSS + TypeScript built with [Vite](https://vitejs.dev/)
- `/static` - static JS bundle for easy prototyping and CMS/static style build systems
- `/angular` (coming soon) - Angular app built with [Angular CLI ](https://angular.io/cli)
- `/react` (coming soon) -  React app built with [Create React App](https://create-react-app.dev/)
- `/vue` (coming soon) -  Vue app built with [Create Vue](https://github.com/vuejs/create-vue)
