# Vertical Flex

Vertical _Flexbox_ based layouts can be achieved with _little to no CSS_ using the `mlv-layout` attribute.

Set `mlv-layout="column"` on a container element and add additional alignment values.

- `mlv-layout="column ..."`


- `align:left`
- `align:right`
- `align:top`
- `align:bottom`
- `align:center`
- `align:horizontal-center`
- `align:vertical-center`
- `align:stretch`
- `align:horizontal-stretch`
- `align:vertical-stretch`
- `align:space-around`
- `align:space-between`
- `align:space-evenly`

Combine `mlv-layout` attribute values to create your desired layout. For example:
```html
<section mlv-layout="column align:vertical-center align:right">
```
