# Typography

The Typography utilities provides a flexible API to apply typography styles explicitly to elements.
This enables full control of the visual styling of an element separate from the HTML semantics.

```html
<p nve-text="display">display</p>
<p nve-text="heading">heading</p>
<p nve-text="body">body</p>
<p nve-text="label">label</p>
<p nve-text="code">cmd+c</p>
```

## Leading Trim

By default all `nve-text` attributes use [leading-trim](https://medium.com/microsoft-design/leading-trim-the-future-of-digital-typesetting-d082d84b202) to remove excesss whitespace between lines. This provides a more precise layout.
Due to the use of `::before` and `::after` to polyfill the `leading-trim` behavior it may be necessary to disable this behavior. This can be done by applying `nve-text="trim:none"`.
