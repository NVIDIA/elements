# Typography

The Typography utilities provides a flexible API to apply typography styles explicitly to elements.
This enables full control of the visual styling of an element separate from the HTML semantics.

```html
<p mlv-text="display">display</p>
<p mlv-text="heading">heading</p>
<p mlv-text="body">body</p>
<p mlv-text="label">label</p>
<p mlv-text="eyebrow">eyebrow</p>
```

<LinkedHeader>Leading Trim</LinkedHeader>

By default all `mlv-text` attributes use [leading-trim](https://medium.com/microsoft-design/leading-trim-the-future-of-digital-typesetting-d082d84b202) to remove excesss whitespace between lines. This provides a more precise layout.
Due to the use of `::before` and `::after` to polyfill the `leading-trim` behavior it may be necessary to disable this behavior. This can be done by applying `mlv-text="trim:none"`.
