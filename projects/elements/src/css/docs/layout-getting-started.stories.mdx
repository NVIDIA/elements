# Layout System

Grid, horizontal and vertical based layouts can be achieved with _little to no CSS_ using the `nve-layout` attribute.

Set `nve-layout="row"` `nve-layout="column"` or `nve-layout="grid"` on a container element and add additional alignment values.

- `nve-layout="row ..."`
- `nve-layout="column ..."`
- `nve-layout="grid ..."`


- `align:left`
- `align:right`
- `align:top`
- `align:bottom ...`

Combine `nve-layout` attribute values to create your desired layout. For example:
```html
<section nve-layout="row align:vertical-center align:right">
```

<LinkedHeader>Layout Gap Spacing</LinkedHeader>

Add the `gap` value to `nve-layout` to set spacing between elements. For example:
```html
<section nve-layout="row gap:md">
```
```html
<section nve-layout="grid gap:md">
```

The following "t-shirt" spacing values are available:

- `gap:xxxs`
- `gap:xxs`
- `gap:xs`
- `gap:sm`
- `gap:md`
- `gap:lg`
- `gap:xl`
- `gap:xxl`
- `gap:xxxl`

<LinkedHeader>Layout Padding</LinkedHeader>


Add the `pad` value to `nve-layout` to set spacing between elements. For example:
```html
<section nve-layout="row pad:md">
```
```html
<section nve-layout="grid pad:md">
```

The following "t-shirt" spacing values are available:

- `pad:xxxs`
- `pad:xxs`
- `pad:xs`
- `pad:sm`
- `pad:md`
- `pad:lg`
- `pad:xl`
- `pad:xxl`
- `pad:xxxl`

By default padding will apply to all 4 sides of container, to specify padding on a single side use:

- `pad-top:md`
- `pad-right:md`
- `pad-bottom:md`
- `pad-left:md`

```html
<section nve-layout="row pad-left:md">
```

<LinkedHeader>Full width/height on container</LinkedHeader>

For convenience a `nve-layout="grow"` option can be set to give the container `100% width & height`.
This is often useful for giving the root element of a page full height.

```html
<section nve-layout="grow">
```
