# Layout System

Grid, horizontal and vertical based layouts can be achieved with _little to no CSS_ using the `mlv-layout` attribute.

Set `mlv-layout="row"` `mlv-layout="column"` or `mlv-layout="grid"` on a container element and add additional alignment values.

- `mlv-layout="row ..."`
- `mlv-layout="column ..."`
- `mlv-layout="grid ..."`


- `align:left`
- `align:right`
- `align:top`
- `align:bottom ...`

Combine `mlv-layout` attribute values to create your desired layout. For example:
```html
<section mlv-layout="row align:vertical-center align:right">
```

## Layout Gap Spacing

Add the `gap` value to `mlv-layout` to set spacing between elements. For example:
```html
<section mlv-layout="row gap:md">

<section mlv-layout="grid gap:md">
```

The following "t-shirt" spacing values are available:

- `gap:xxxs`
- `gap:xxs`
- `gap:xs`
- `gap:sm`
- `gap:md`
- `gap:lg`
- `gap:xl`
- `gap:xxl`
- `gap:xxxl`

## Layout Padding


Add the `pad` value to `mlv-layout` to set spacing between elements. For example:
```html
<section mlv-layout="row pad:md">

<section mlv-layout="grid pad:md">
```

The following "t-shirt" spacing values are available:

- `pad:xxxs`
- `pad:xxs`
- `pad:xs`
- `pad:sm`
- `pad:md`
- `pad:lg`
- `pad:xl`
- `pad:xxl`
- `pad:xxxl`

By default padding will apply to all 4 sides of container, to specify padding on a single side use:

- `pad-top:md`
- `pad-right:md`
- `pad-bottom:md`
- `pad-left:md`

Or alternatively use the short hand to just pad the x and y axes.

- `pad-x:md`
- `pad-y:md`

```html
<section mlv-layout="row pad-left:md">
```

## Full width/height on container

For convenience a `mlv-layout="full"` option can be set to give the container 100% width & height.
This is often useful for giving the root element of a page full height.

```html
<section mlv-layout="full">
```
