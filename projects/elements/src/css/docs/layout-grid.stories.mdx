# Grid

CSS _Grid_ based layouts can be achieved with _little to no CSS_ using the `mlv-layout` attribute.
The system is based on a 12 column layout.

Set `mlv-layout="grid"` on a container element and add additional item spanning (`span-items:...`) and `gap:...` values.

- `mlv-layout="grid gap:md span-items:1"`
- `mlv-layout="grid gap:md span-items:2"`
- `mlv-layout="grid gap:md span-items:3"`
- `mlv-layout="grid gap:md span-items:..."`
- `mlv-layout="grid gap:md span-items:12"`

Or specify column spanning (`span:...`) on the individual items (1-12):

- `mlv-layout="span:1"`
- `mlv-layout="span:2"`
- `mlv-layout="span:3"`
- `mlv-layout="span:..."`
- `mlv-layout="span:12"`

Similar to the Horizontal and Vertical _Flexbox_ based system you can also add alignment values:

- `align:left`
- `align:right`
- `align:top`
- `align:bottom`
- `align:center`
- `align:horizontal-center`
- `align:vertical-center`
- `align:stretch`
- `align:horizontal-stretch`
- `align:vertical-stretch`

<LinkedHeader>Grid Layout Examples</LinkedHeader>

```html
<!--  Specify column spanning on the parent -->
<section mlv-layout="grid span-items:3">
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
</section>

<!--  Specify column spanning on the children -->
<section mlv-layout="grid gap:md">
  <mlv-card mlv-layout="span:6"></mlv-card>
  <mlv-card mlv-layout="span:6"></mlv-card>
  <mlv-card mlv-layout="span:4"></mlv-card>
  <mlv-card mlv-layout="span:4"></mlv-card>
  <mlv-card mlv-layout="span:4"></mlv-card>
  <mlv-card mlv-layout="span:4"></mlv-card>
</section>

<!--  Grid with alignment options -->
<section mlv-layout="grid gap:md align:center">
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
  <mlv-card></mlv-card>
</section>
```
