# Icon

<element-metrics tag="nve-icon"></element-metrics>

The Iconography system is based on exposing an SVG based icon library to a the `<nve-icon>` element.

See the searchable [Interactive Icon Catalog](./?path=/story/foundations-iconography-documentation--page)

<LinkedHeader>Installation</LinkedHeader>

```typescript
import '@elements/elements/icon/define.js';
```

```html
<nve-icon name="user"></nve-icon>
```
