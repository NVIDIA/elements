# Icon

<element-metrics tag="mlv-icon"></element-metrics>

The Iconography system is based on exposing an SVG based icon library to a the `<mlv-icon>` element.
See the searchable [Interactive Icon Catalog](./?path=/docs/foundations-iconography-documentation--docs)

## Installation

```typescript
import '@elements/elements/icon/define.js';
```

```html
<mlv-icon name="person"></mlv-icon>
```
