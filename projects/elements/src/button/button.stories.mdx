## Linked Buttons

Anchors can wrap button instances, however its recommended to slot the anchor into the button.

```html
<mlv-button><a href="#">default</a></mlv-button> <!-- do -->
<a href="#"><mlv-button>default</mlv-button></a> <!-- don't -->
```

If you need to slot an icon in a linked button, put `<mlv-icon>` adjacent to the anchor tag.

```html
<mlv-button><a href="#">default</a> <mlv-icon name="reset"></mlv-icon></mlv-button> <!-- do -->
```
