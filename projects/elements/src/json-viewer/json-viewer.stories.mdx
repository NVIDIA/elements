## Installation

```typescript

```

```html
  {
    "tasks": [
      { "id": 1, "status": "success", "date": "08/03/2023 14:22:44", "url": "https://elements.nvidia.com/av/ingestion/uploads", "message": "task completed" },
      { "id": 2, "status": "failed", "date": "08/09/2023 14:22:44", "url": "https://elements.nvidia.com/av/ingestion/uploads", "message": "Error: `c.request()` failed on:\n\nhttps://elements.nvidia.com/service/v1/\n\nThe response we received from your web server was:\n\n  > 401: Unauthorized\n" }
    ]
  }
</mlv-json-viewer>
```
