# Data Grid Placeholder

<element-metrics tag="mlv-grid-placeholder"></element-metrics>

Placeholder can be used to display a message when data is being loaded for the grid.

```html
<mlv-grid style="--scroll-height: 333px">
  <mlv-grid-header>
    <mlv-grid-column></mlv-grid-column>
  </mlv-grid-header>
  <mlv-grid-placeholder>
    Loading...
  </mlv-grid-placeholder>
</mlv-grid>
```
