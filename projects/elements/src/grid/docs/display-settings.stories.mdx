# Data Grid Display Settings

<Metrics tag="mlv-grid-column"></Metrics>

Display settings are placed above the data grid leveraging dropdowns to provide various configuration options.
Columns can be hidden via the `hidden` attribute or conditionaly rendered in DOM. Display settings can include
column visibility or advanced filtering options.

```html
<section mlv-layout="row gap:md align:stretch">
  <mlv-panel expanded closable>
    ...
  </mlv-panel>

  <div mlv-layout="column gap:md">
    <mlv-button>display settings</mlv-button>
    <mlv-grid>
      ...
    </mlv-grid>
  </div>
</section>
```
