# Data Grid Container

<Metrics tag="mlv-grid"></Metrics>

The `container` property/attribute enables the grid container to be altered for use cases such as nesting within another component like cards or full screen widths.

```html
<mlv-grid container="fill">
  ...
</mlv-grid>
```
