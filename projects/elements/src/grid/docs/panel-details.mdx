# Data Grid Panel Details

<element-metrics tag="mlv-grid"></element-metrics>

Use a right aligned [panel](./?path=/docs/elements-panel-documentation--docs) when displaying advanced filtering or display settings for the grid.
Item detail panels should be open using a action button placed at the end of the grid row.

```html
<section mlv-layout="row gap:md align:stretch">
  <mlv-grid>
    <mlv-grid-header>
      <mlv-grid-column>Column 1</mlv-grid-column>
      <mlv-grid-column>Column 2</mlv-grid-column>
      <mlv-grid-column>Column 3</mlv-grid-column>
      <mlv-grid-column width="max-content" aria-label="details"></mlv-grid-column>
    </mlv-grid-header>

    <mlv-grid-row>
      <mlv-grid-cell>Cell 1-1</mlv-grid-cell>
      <mlv-grid-cell>Cell 1-2</mlv-grid-cell>
      <mlv-grid-cell>Cell 1-3</mlv-grid-cell>
      <mlv-grid-cell>
        <mlv-icon-button container="flat" icon-name="additional-actions" aria-label="view item 1"></mlv-icon-button>
      </mlv-grid-cell>
    </mlv-grid-row>
  </mlv-grid>

  <mlv-panel expanded closable></mlv-panel>
</section>
```
