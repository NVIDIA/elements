# Data Grid Column Width

<Metrics tag="mlv-grid-column"></Metrics>

Columns can individually have set widths using the `width` property or attribute. If the total width of all columns
is less than the grid width then the last column will fill the remaining space.

```html
<mlv-grid>
  <mlv-grid-header>
    <mlv-grid-column width="200px">Column 1</mlv-grid-column>
    <mlv-grid-column width="200px">Column 2</mlv-grid-column>
    <mlv-grid-column width="200px">Column 3</mlv-grid-column>
    <mlv-grid-column width="200px">Column 4</mlv-grid-column>
  </mlv-grid-header>
  <mlv-grid-row>
    <mlv-grid-cell>cell 1-1</mlv-grid-cell>
    <mlv-grid-cell>cell 1-2</mlv-grid-cell>
    <mlv-grid-cell>cell 1-3</mlv-grid-cell>
    <mlv-grid-cell>cell 1-4</mlv-grid-cell>
  </mlv-grid-row>
  ...
</mlv-grid>
```
