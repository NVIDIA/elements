# Data Grid Single Select

<Metrics tag="mlv-grid-row"></Metrics>

Single selection enables users to use a radio list association to select one item at a time.
To enable single select, place a `mlv-radio` input as the first grid cell of each row. Set the `name`
attribute on each radio to ensure they associate to the same radio group.

```html
<mlv-grid>
  <mlv-grid-header>
    <mlv-grid-column width="max-content" aria-label="single selection"></mlv-grid-column>
    <mlv-grid-column>Column</mlv-grid-column>
    ...
  </mlv-grid-header>
  <mlv-grid-row selected>
    <mlv-grid-cell>
      <mlv-radio>
        <input type="radio" name="single-select" checked aria-label="select row 1">
      </mlv-radio>
    </mlv-grid-cell>
    <mlv-grid-cell>Cell</mlv-grid-cell>
    ...
  </mlv-grid-row>
  ...
</mlv-grid>
```
