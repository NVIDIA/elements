# Data Grid Placeholder

<element-metrics tag="mlv-grid-placeholder"></element-metrics>

Placeholder can be used to display a message when data is being loaded for the grid or empty states for datasets.

```html
<mlv-grid style="min-height: 400px">
  <mlv-grid-header>
    <mlv-grid-column></mlv-grid-column>
  </mlv-grid-header>
  <mlv-grid-placeholder>
    <mlv-progress-ring status="accent" size="lg"></mlv-progress-ring>
  </mlv-grid-placeholder>
</mlv-grid>
```
