# Data Grid Placeholder

<nve-api-summary tag="nve-grid-placeholder"></nve-api-summary>

Placeholder can be used to display a message when data is being loaded for the grid or empty states for datasets.

```html
<nve-grid style="min-height: 400px">
  <nve-grid-header>
    <nve-grid-column></nve-grid-column>
  </nve-grid-header>
  <nve-grid-placeholder>
    <nve-progress-ring status="accent" size="lg"></nve-progress-ring>
  </nve-grid-placeholder>
</nve-grid>
```
