# Data Grid Column Action

<element-metrics tag="nve-grid-column"></element-metrics>

Column actions can be created by using the `mlv-icon-button` to trigger dropdowns or panels to reveal additional actions to the user.

```html
<nve-grid>
  <nve-grid-header>
    <nve-grid-column>
      Column 1 <nve-icon-button slot="actions"></nve-icon-button>
    </nve-grid-column>
    <nve-grid-column>Column 2</nve-grid-column>
    <nve-grid-column>Column 3</nve-grid-column>
    <nve-grid-column>Column 4</nve-grid-column>
  </nve-grid-header>
    <nve-grid-row>
      ...
    </nve-grid-row>
  ...
</nve-grid>
```
