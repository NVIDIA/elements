# Panel Grid

<element-metrics tag="mlv-grid"></element-metrics>

Panel Grid can be used to display key value type data sets for details of a given item in a collection.

```html
<main mlv-layout="row gap:sm full">
  <section mlv-layout="column gap:md full align:stretch">
    page content
  </section>
  <mlv-panel expanded>
    <mlv-grid container="flat" stripe>
      ...
    </mlv-grid>
  </mlv-panel>
</main>
```
