# Data Grid Column Alignment

<Metrics tag="mlv-grid-column"></Metrics>

Column alignments can be controlled using the `column-align` attribute or `columnAlign` property. Column alignment can be individually controlled.

```html
<mlv-grid>
  <mlv-grid-header>
    <mlv-grid-column column-align="left">Column 1</mlv-grid-column>
    <mlv-grid-column column-align="right">Column 2</mlv-grid-column>
    <mlv-grid-column column-align="center">Column 3</mlv-grid-column>
    <mlv-grid-column>Column 4</mlv-grid-column>
  </mlv-grid-header>
  <mlv-grid-row>
    <mlv-grid-cell>cell 1-1</mlv-grid-cell>
    <mlv-grid-cell>cell 1-2</mlv-grid-cell>
    <mlv-grid-cell>cell 1-3</mlv-grid-cell>
    <mlv-grid-cell>cell 1-4</mlv-grid-cell>
  </mlv-grid-row>
  ...
</mlv-grid>
```
