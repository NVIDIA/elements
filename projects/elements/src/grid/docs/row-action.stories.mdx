# Data Grid Row Action

<element-metrics tag="mlv-grid-row"></element-metrics>

Row actions enable addtional user actions specific to a given row. Place a `mlv-icon-button` at the end
of the grid row for actions.

```html
<mlv-grid>
  <mlv-grid-header>
    <mlv-grid-column>Column 1</mlv-grid-column>
    <mlv-grid-column>Column 2</mlv-grid-column>
    <mlv-grid-column>Column 3</mlv-grid-column>
    <mlv-grid-column width="max-content" position="fixed" aria-label="additonal actions"></mlv-grid-column>
  </mlv-grid-header>
  ...
  <mlv-grid-row>
    <mlv-grid-cell>Cell 1-1</mlv-grid-cell>
    <mlv-grid-cell>Cell 1-2</mlv-grid-cell>
    <mlv-grid-cell>Cell 1-3</mlv-grid-cell>
    <mlv-grid-cell>
      <mlv-icon-button interaction="flat" icon-name="additional-actions" aria-label="row 1 actions"></mlv-icon-button>
    </mlv-grid-cell>
  </mlv-grid-row>
  ...
</mlv-grid>
```
