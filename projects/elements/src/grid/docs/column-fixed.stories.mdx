# Data Grid Column Fixed

<Metrics tag="mlv-grid-column"></Metrics>

Columns positioned to a fixed location within the grid preventing the column from scrolling when the grid has horizontal scroll. Use the `position="fixed"` attribute to position the column.

```html
<mlv-grid>
  <mlv-grid-header>
    <mlv-grid-column position="fixed">Column 1</mlv-grid-column>
    <mlv-grid-column>Column 2</mlv-grid-column>
    <mlv-grid-column>Column 3</mlv-grid-column>
    <mlv-grid-column>Column 4</mlv-grid-column>
  </mlv-grid-header>
  <mlv-grid-row>
    <mlv-grid-cell>cell 1-1</mlv-grid-cell>
    <mlv-grid-cell>cell 1-2</mlv-grid-cell>
    <mlv-grid-cell>cell 1-3</mlv-grid-cell>
    <mlv-grid-cell>cell 1-4</mlv-grid-cell>
  </mlv-grid-row>
  ...
</mlv-grid>
```
