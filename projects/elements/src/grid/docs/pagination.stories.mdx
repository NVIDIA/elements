# Data Grid Pagination

<element-metrics tag="mlv-grid"></element-metrics>

The pagination pattern should be used when working with large data sets that need to be
incrementally loaded or filtered for performance or usablity.

```html
<mlv-grid>
  ...

  <mlv-grid-footer>
    <mlv-pagination value="1" items="100" step="10"></mlv-pagination>
  </mlv-grid-footer>
</mlv-grid>
```
