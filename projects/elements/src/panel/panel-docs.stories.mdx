# Panels

Panels provide a way to make detail content available alongside the primary page layout,
in a way that is accessible and minimizable.

Side panel is composed of a `<nve-panel>` container element and several sub-components with default styling.
The following sub-components can be used within `nve-panel` to achieve standard panel layouts:

- `<nve-panel-content>`
- `<nve-panel-footer>`

Panels will always be minimizable, either by closing or collapsing. The proper close/collapse icon button will display depending on the
whether or not `closable` is set. By default `closable` is off and the panel's minimizing function is to collapse to just an icon button.

Additionally, `<nve-panel>` can be have it's `side` property set to `left` or `right` to get the proper collapse/expand icon and animation
depending on which side of the page you are using the panel.

An slot for `action-icon` is exposed allowing you to render an additional button in the header.

<LinkedHeader>Installation</LinkedHeader>

```typescript
import '@elements/elements/panel/define.js';
```

```html
<nve-panel collapsable>
  <nve-panel-content>
    Panel Content
  </nve-panel-content>
</nve-panel>
```
