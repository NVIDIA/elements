# Control Actions

Text type controls support prefix and suffix labeling as well as action buttons.

<LinkedHeader>Installation</LinkedHeader>

```typescript
import '@elements/elements/input/define.js';
```

```html
<nve-input>
  <label>label</label>
  <input type="text" />
  <nve-icon-button icon-name="cancel" interaction="flat" aria-label="clear"></nve-icon-button>
</nve-input>
```
