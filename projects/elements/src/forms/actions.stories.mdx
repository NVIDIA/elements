# Control Actions

Text type controls support prefix and suffix labeling as well as action buttons.

## Installation

```typescript
import '@elements/elements/input/define.js';
```

```html
<mlv-input>
  <label>label</label>
  <input type="text" />
  <mlv-icon-button icon-name="cancel" container="flat" aria-label="clear"></mlv-icon-button>
</mlv-input>
```
