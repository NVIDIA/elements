# Control Actions

Text type controls support prefix and suffix labeling as well as action buttons.

<LinkedHeader>Installation</LinkedHeader>

```typescript
import '@elements/elements/input/define.js';
```

```html
<mlv-input>
  <label>label</label>
  <input type="text" />
  <mlv-icon-button icon-name="cancel" interaction="ghost" aria-label="clear"></mlv-icon-button>
</mlv-input>
```
