# Radio

[MDN Documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio)

## Installation

```typescript
import '@elements/elements/radio/define.js';
```

```html
<mlv-radio-group>
  <label>label</label>
  <mlv-radio>
    <label>radio 1</label>
    <input type="radio" checked />
  </mlv-radio>

  <mlv-radio>
    <label>radio 2</label>
    <input type="radio" />
  </mlv-radio>

  <mlv-radio>
    <label>radio 3</label>
    <input type="radio" />
  </mlv-radio>
  <mlv-control-message>message</mlv-control-message>
</mlv-radio-group>
```
