
<LinkedHeader>Versioning</LinkedHeader>

The `@elements/elements` pacakge follows [semver](https://semver.org/) and is currently pre 1.0. Changes can be found in the [Changelog](./?path=/story/about-changelog--page). A debug utility is available
on the global window object to help identify the active versions being used at runtime. This log will list out all the registered elements and active versions.

```typescript
window.MLV_ELEMENTS.debug()
```

```json
{

  "elementRegistry": {
```
