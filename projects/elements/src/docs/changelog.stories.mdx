<LinkedHeader>0.5.0 - 11-22-2022</LinkedHeader>

### Features
- Added alpha stage dialog element
- Added alpha stage dropdown element
- Added alpha stage layer/interaction tokens

### Fixes
- Fix invalid state on date input not resetting on blur
- Fix placeholder color for textarea
- Textarea correctly overrides initial min-height when using the `row` attr.

### Breaking
- Removal of interaction state tokens. Interaction background and color tokens remain however the alternate states (hover, active) are removed.
  Background contrast is now used to determine the interaction state. See [Interaction states](./?path=/story/foundations-tokens-interactions--page) for more information.
- Fix: Theme base styles such as font and background colors are only applied when `mlv-theme` attr is placed on the `html`, `body` element or with `mlv-theme="root"`.
  This allows for more flexibility when using testing themes without overiding the nested element style properties.
- Rename internals and `--mlv-sys-layer-popup-*` to `--mlv-sys-layer-popover-*` to align with [popover spec](https://github.com/openui/open-ui/issues/627#issuecomment-1293918937)

<LinkedHeader>0.2.1 - 08-09-2022</LinkedHeader>

### Features
- Now supports individual element imports for better performance
  ```javascript
  import '@elements/elements/button/define.js';
  ```
  <br />
- Added `mlv-button-icon` element
  ```html
  <mlv-icon-button interaction="emphasize" icon-name="analytics"></mlv-icon-button>
  ```
  <br />
- Build artifact redesigned to reduce side effects in build
- Added MDX docs to Storybook
- Added Lit testing utils for element unit testing
- Updated Storybook to version 6.5
- `mlv-button` and `mlv-button-icon` now based on base button class w/ submit and aria support
    - Refactored with button behavior Lit controllers
- Added beta release of design token integration into elements

### Fixes
- `mlv-button` and `mlv-button-icon` now support full HTML Form submit events
- `mlv-button` and `mlv-button-icon` now support keyboard interactions (space/enter)

### Breaking Changes
- Button `variant` has been replaced with button `interaction` to align with design
- Removed `icon` property on `mlv-icon`
- Refactored `mlv-card` to a composition / slot based API for more flexibility
    - Adds `mlv-card-header`
    - Adds `mlv-card-footer`
