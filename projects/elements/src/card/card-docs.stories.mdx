# Card

Card is composed of a `nve-card` container element and several sub-components with default styling.
The following sub-components can be used within `nve-card` to achieve standard card layouts:

- `nve-card-header`
- `nve-card-content`
- `nve-card-footer`

Content slotted directly under `nve-card` is supported for full flexibility of content.

## Installation

```typescript
import '@elements/elements/card/define.js';
```

```html
<nve-card>
  <nve-card-content>
    Card Content
  </nve-card-content>
</nve-card>
```

## Card with Content Layout

Layout system is supported when <a href="./?path=/story/layout-getting-started--page">nve-layout</a> attribute
is used on the `nve-card-content` element:

```html
<nve-card>
  <nve-card-content nve-layout="row align:space-around">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
  </nve-card-content>
</nve-card>
```

## Card with Header and Footer

Combining `nve-card-header`, `nve-card-content` & `nve-card-footer` will yield a fully styled card.
Note how `nve-card-content` will vertically stretch to fill the card size and the footer will align
to the bottom of the card.
