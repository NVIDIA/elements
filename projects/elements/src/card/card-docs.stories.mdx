# Card

Card is composed of a `mlv-card` container element and several sub-components with default styling.
The following sub-components can be used within `mlv-card` to achieve standard card layouts:

- `mlv-card-header`
- `mlv-card-content`
- `mlv-card-footer`

Content slotted directly under `mlv-card` is supported for full flexibility of content.

<LinkedHeader>Installation</LinkedHeader>

```typescript
import '@elements/elements/card/define.js';
```

```html
<mlv-card>
  <mlv-card-content>
    Card Content
  </mlv-card-content>
</mlv-card>
```

<LinkedHeader>Card with Content Layout</LinkedHeader>

Layout system is supported when <a href="./?path=/story/layout-getting-started--page">mlv-layout</a> attribute
is used on the `mlv-card-content` element:

```html
<mlv-card>
  <mlv-card-content mlv-layout="row align:space-around">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
  </mlv-card-content>
</mlv-card>
```

<LinkedHeader>Card with Header and Footer</LinkedHeader>

Combining `mlv-card-header`, `mlv-card-content` & `mlv-card-footer` will yield a fully styled card.
Note how `mlv-card-content` will vertically stretch to fill the card size and the footer will align
to the bottom of the card.
