# Registration

## Tag Name Convention
Custom Element tags are globally registered to the browser so elements must be scoped to prevent collisions.

- mlv-* Maglev Elements
- mlv-ide-* Maglev Shell
- [plugin/app abbrev]-* MagLev Plugins/Apps
