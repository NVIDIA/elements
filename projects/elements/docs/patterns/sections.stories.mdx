# Section Patterns

Section patterns demonstrate the composition of `MagLev Elements` with proper utilization of the attribute based
[Layout System](./?path=/docs/foundations-layout-documentation--docs). <br />
Example patterns include header component and a floating action button based feedback widget.

All of the example are built with around **~100 lines of HTML and nearly zero CSS**.
