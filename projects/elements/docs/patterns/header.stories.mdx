# Page Header Patterns

Intended for use at the top of pages, but below the [App Header](./?path=/docs/elements-app-header-main-page-documentation--docs), the following
combinations of components are recommended patterns for composing _Page Headers_.


## Main Page Headers

Page headers used for _main content pages_ should use `<nve-card container="flat">` cotaining sections for a `<h1>` page title, and sections for
action buttons, breadcrumbs, and optional metadata.
