To use Elements in Vue follow the [instalation getting started](?path=/docs/about-getting-started--docs) steps. Once complete
elements can be imported and used within Vue SFC files.

```javascript
import '@elements/elements/alert-group/define.js';
```

Once added, properties and events can be used via the standard Vue template syntax.

```html
<!--
- status - HTML attribute
- :closable - can update via attributes or JavaScript property binding
- @close - event listener binding for 'close' custom event
-->

<mlv-alert-group status="success">
  <mlv-alert :closable="true" @close="closeAlert">hello there!</mlv-alert>
</mlv-alert-group>
```
