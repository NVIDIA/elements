# React Integration

<mlv-alert status="warning">
  While Elements can be used in React we recommend alternatives such as <a mlv-text="link" href="https://preactjs.com">Preact</a> due to <a mlv-text="link" href="https://custom-elements-everywhere.com/">React incompatibilities</a> with standard Web APIs.
</mlv-alert>

To use Maglev Elements in [React](https://reactjs.org/) follow the [instalation getting started](?path=/docs/about-getting-started--docs) steps. Once complete
elements can be imported and used within React jsx and tsx files. The `@lit/react` package must be used to enable React compatibility with Custom Elements.
This package will wrap elements into a React component mapping the standard events and properties in a way that React can understand.

```tsx
import * as React from 'react';
import { createComponent } from '@lit/react';
import { Dialog } from '@elements/elements/dialog/index.js';
import '@elements/elements/dialog/define.js';

export const MlvDialog = createComponent({ react: React, tagName: 'mlv-dialog', elementClass: Dialog, events: { onclose: 'close' } });
```

The `MlvDialog` React component can now be used within a React application.

```tsx
// - closable - HTML attribute
// - hidden - can update via attributes or JavaScript property binding
// - onclose - event listener binding for 'close' custom event

<MlvDialog closable modal hidden={!showDialog} onclose={() => setShowDialog(false)}>
  ...
</MlvDialog>
```

To learn more about the `@lit/react` package, please visit the [documentation](https://github.com/lit/lit/blob/main/packages/react/README.md).
View a working [demo here](https://elements-stage.nvidia.com/ui/elements-playground/d445ba8d-a2c9-41c4-bea9-18a6bbf27100?layout=vertical-split&theme=light&file=index.tsx).
