To use Elements in [SolidJS](https://www.solidjs.com/) follow the [instalation getting started](?path=/docs/about-getting-started--docs) steps. Once complete
elements can be imported and used within SolidJS jsx and tsx files.

```javascript
import '@elements/elements/alert-group/define.js';
```

Once added, properties and events can be used via the standard template syntax.

```jsx
// - status - HTML attribute
// - closable - can update via attributes or JavaScript property binding
// - onclose - event listener binding for 'close' custom event

<mlv-alert-group status="success">
  <mlv-alert closable={isClosable} onclose={(e) => closeAlert()}>hello there!</mlv-alert>
</mlv-alert-group>
```


To add TypeScript types to your TSX files add the element to the `IntrinsicElements` interface.

```typescript
// types.d.ts
import type { MlvAlert, MlvAlertGroup } from '@elements/elements/alert-group';

type CustomEvents<K extends string> = { [key in K]: (event: CustomEvent) => void; };
type CustomElement<T, K extends string = ""> = Partial<T & { children: any } & CustomEvents<`on${K}`>>;

declare module 'solid-js' {
  namespace JSX {
    interface IntrinsicElements {
      ['mlv-alert']: CustomElement<MlvAlert>;
      ['mlv-alert-group']: CustomElement<MlvAlertGroup>;
    }
  }
}
```
