# Themes

Themes are built leveraging CSS Custom Properties. Themes are applied by setting the `mlv-theme` attribute on the host `html` element.

```css
@import '@elements/elements/dist/index.css';
```

```html
<html lang="en" mlv-theme="dark">
```

<mlv-alert status="warning">
  If your application cannot access root nodes like html or body then apply mlv-theme="root" to set the root background colors and fonts.
</mlv-alert>

<br /><br />

## Theme Options

Currently there are the following themes:

- Light (default)
- Dark
- High Contrast
- Reduced Motion (alpha)
- Compact (alpha)

Themes can be applied dynamically based on user preferences.

```javascript
 if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
   document.body.setAttribute('mlv-theme', 'dark');
 }

 if (window.matchMedia('(-ms-high-contrast: active)').matches || window.matchMedia('(forced-colors: active)').matches) {
   document.body.setAttribute('mlv-theme', 'high-contrast');
 }

 if (window.matchMedia('(prefers-reduced-motion: reduce)').matches) {
   document.body.setAttribute('mlv-theme', 'reduced-motion');
 }
```
