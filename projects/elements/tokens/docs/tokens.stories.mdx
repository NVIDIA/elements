# Design Tokens (beta)

Design Tokens and Themes provide the foundation for the look and feel of the MagLev Platform.
Design tokens are organized in layers to provide semantic and purposeful values
for UI design.

### reference tokens (foundation, `--nve-ref-*`)

  - color
  - space
  - size
  - type

### system tokens (semantic, `--nve-sys-*`)

  - status
  - layers (canvas, containers, overlays, popovers)
  - interactions (default, emphasized, destructive)

### themes (token overrides)

  - dark theme
  - compact theme

### components (consumers)

  - use tokens in CSS, defaulting to system tokens when possible
