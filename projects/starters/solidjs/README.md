# NVIDIA Elements + SolidJS Starter

An interactive SolidJS starter for the [NVIDIA Elements design system](https://nvidia.github.io/elements/), built with Vite.

## Getting started

```shell
npm install
npm run dev
```

Open `http://localhost:3000/` to view the application.

## Tasks

| Command           | Description                        |
| ----------------- | ---------------------------------- |
| `npm run dev`     | Start the local development server |
| `npm run build`   | Create a production build          |
| `npm run preview` | Preview the production build       |
| `npm run lint`    | Lint the Solid source              |

See the [Solid documentation](https://docs.solidjs.com/) and [Vite documentation](https://vite.dev/) for more information.
