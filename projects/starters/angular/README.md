# NVIDIA Elements + Angular Starter

An interactive Angular starter for the [NVIDIA Elements design system](https://nvidia.github.io/elements/).

## Getting started

```shell
npm install
npm run dev
```

Open `http://localhost:4200/` to view the application. The development server reloads when you change a source file.

## Tasks

| Command         | Description                       |
| --------------- | --------------------------------- |
| `npm run dev`   | Start the local development server |
| `npm run build` | Create a production build          |
| `npm test`      | Run the Vitest unit tests           |
| `npm run lint`  | Lint the application source         |

See the [Angular CLI documentation](https://angular.dev/tools/cli) for more commands.
