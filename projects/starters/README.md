# Starter Applications

This directory contains a suite of standardized starter apps for kickstarting and using Elements in a variety of Web technologies.

- `/angular` - Angular app built with [Angular CLI](https://angular.io/cli)
- `/buildless` - Plain HTML with import maps, no build step required.
- `/extensions` - Build setup for highly reusable and resiliant Web Components.
- `/mpa` - Multi Page App setup for when you just need something simple.
- `/nextjs` - Experimental NextJS demo.
- `/react` - React app built with [Create React App](https://create-react-app.dev/)
- `/solidjs` - [SolidJS](https://www.solidjs.com/) app built with [Vite](https://vitejs.dev/)
- `/static` - No tooling bundle and no performance optimizations
- `/typescript` - HTML + CSS + TypeScript built with [Vite](https://vitejs.dev/)
- `/vue` - Vue app built with [Create Vue](https://github.com/vuejs/create-vue)
