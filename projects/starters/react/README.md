# NVIDIA Elements + React Starter

An interactive React starter for the [NVIDIA Elements design system](https://nvidia.github.io/elements/), built with Vite.

## Getting started

```shell
npm install
npm run dev
```

Open `http://localhost:5173/` to view the application.

## Tasks

| Command           | Description                        |
| ----------------- | ---------------------------------- |
| `npm run dev`     | Start the local development server |
| `npm run build`   | Create a production build          |
| `npm run preview` | Preview the production build       |
| `npm run lint`    | Lint the React source              |

See the [React documentation](https://react.dev/) and [Vite documentation](https://vite.dev/) for more information.
