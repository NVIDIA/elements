# NVIDIA Elements + Vue Starter

An interactive Vue starter for the [NVIDIA Elements design system](https://nvidia.github.io/elements/), built with Vite.

## Recommended editor setup

Use [Visual Studio Code](https://code.visualstudio.com/) with the [Vue - Official](https://marketplace.visualstudio.com/items?itemName=Vue.volar) extension.

## Getting started

```shell
npm install
npm run dev
```

Open `http://localhost:5173/` to view the application.

## Tasks

| Command           | Description                           |
| ----------------- | ------------------------------------- |
| `npm run dev`     | Start the local development server    |
| `npm run build`   | Type-check and create a production build |
| `npm run preview` | Preview the production build          |
| `npm run lint`    | Lint the Vue and TypeScript source     |

See the [Vue TypeScript guide](https://vuejs.org/guide/typescript/overview.html) and [Vite documentation](https://vite.dev/) for more information.
