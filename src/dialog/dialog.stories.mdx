# Dialog

[W3 ARIA Pattern Documentation](https://www.w3.org/WAI/ARIA/apg/patterns/dialogmodal/)

Dialog is a component that appears above main content. A modal dialog is used to
display critical information that requires users attention. Use `hidden` to show or hide the dialog.

## Installation

```typescript
import '@elements/elements/dialog/define.js';
```

```html
<nve-dialog>
  hello there
</nve-dialog>
```
