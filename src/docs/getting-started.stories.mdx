# Maglev Elements

## Foundational building blocks for Maglev UI Patterns.

## Installation

To install the Maglev UI Elements, run the following command:

```bash
pnpm install @elements/elements
```

Once installed import the global CSS into your project.

```css
@import '@elements/elements/index.css';
```

Elements can be imported/loaded via JavaScript imports and made available to use in your HTML templates.

```javascript
import '@elements/elements/button/define.js';
```

```html
<mlv-button>hello there</mlv-button>
```

<mlv-button interaction="emphasize">hello there</mlv-button>
