# Maglev Elements

## Foundational building blocks for Maglev UI Patterns.

## Installation

To install the Maglev UI Elements, run the following command:

```bash
pnpm install @elements/elements
```

Once installed import the global CSS into your project.

```css
@import '@elements/elements/dist/index.css';
```

Elements can be imported/loaded via JavaScript imports and made available to use in your HTML templates.

```javascript
import '@elements/elements/button/define.js';
```

```html
<nve-button>hello there</nve-button>
```

<nve-button interaction="emphasize">hello there</nve-button>

Alternatively, you can import the entire @elements/elements library, but will incur a larger bundle size.
```javascript
import '@elements/elements';
```

Angular applications have a couple additional <a href="./?path=/story/about-angular--page">installation steps</a>.
