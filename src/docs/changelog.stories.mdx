# Changelog

## v0.3.0

### Features
- Added Horizontal and Vertical Layout System
- Adds MVP of Grid Layout System
- Initial launch of V1 of the Forms Elements

## v0.2.1

### Features
- Now supports individual element imports for better performance
  ```javascript
  import '@elements/elements/button/define.js';
  ```
  <br />
- Added `nve-button-icon` element
  ```html
  <nve-icon-button interaction="emphasize" icon-name="analytics"></nve-icon-button>
  ```
  <br />
- Build artifact redesigned to reduce side effects in build
- Added MDX docs to Storybook
- Added Lit testing utils for element unit testing
- Updated Storybook to version 6.5
- `nve-button` and `nve-button-icon` now based on base button class w/ submit and aria support
    - Refactored with button behavior Lit controllers
- Added beta release of design token integration into elements

### Fixes
- `nve-button` and `nve-button-icon` now support full HTML Form submit events
- `nve-button` and `nve-button-icon` now support keyboard interactions (space/enter)

### Breaking Changes
- Button `variant` has been replaced with button `interaction` to align with design
- Removed `icon` property on `nve-icon`
- Refactored `nve-card` to a composition / slot based API for more flexibility
    - Adds `nve-card-header`
    - Adds `nve-card-footer`

## v0.1.3
- Initial release of alpha version to artifactory
- POC version of Button/Icon/Card elements based in Lit/Vite/Storybook build tooling
