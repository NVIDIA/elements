# Control

## Installation

```typescript
import '@elements/elements/forms/define.js';
```

The `mlv-control` element is the base control that all built in `mlv-*` form controls extend.
The `mlv-control` can be used for generic imput type as well as custom or third party form controls.

## Custom

Custom inputs can be used and leverage the same form layouts and validation statuses as the built in controls.
Place the `mlv-control` attribute on the element that is the custom input. [emoji picker source](https://github.com/nolanlawson/emoji-picker-element).
