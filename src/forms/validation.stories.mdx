# Validation

The `nve-control` supports various validation statuses such as error and success. These statuses can be
toggle by any framework level forms API. However the `nve-control` component does also
support the native HTML Validation API. Defining a Validity State on a `nve-control-message` will
allow messages to be shown conditionally based on the current input validity state.

[MDN Validity State](https://developer.mozilla.org/en-US/docs/Web/API/ValidityState)
