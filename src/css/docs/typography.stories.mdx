# Typography (beta)

The Typography utilities provides a flexible API to apply typography styles explicitly to elements.
This enables full control of the visual styling of an element separate from the HTML semantics.

```html
<p mlv-text="display">display</p>
<p mlv-text="heading">heading</p>
<p mlv-text="body">body</p>
<p mlv-text="label">label</p>
<p mlv-text="eyebrow">eyebrow</p>
```
