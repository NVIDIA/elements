# Status (beta)

Status colors are used in combination of text or ionography to convey status of 
an element or interation. Status type components inlcude `alerts`, `badges`, `tags` and `form control` messages.

- `neutral`
- `info`
- `success`
- `warning`
- `danger`

## Accent Tokens

Accent tokens are used as system wide color accents. Components using accent colors
inlcude checkboxes, radios, range inputs and form controls.
