# Radio

[MDN Documentation](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/input/radio)

## Installation

```typescript
import '@elements/elements/radio/define.js';
```

```html
<nve-radio-group>
  <label>label</label>
  <nve-radio>
    <label>radio 1</label>
    <input type="radio" checked />
  </nve-radio>

  <nve-radio>
    <label>radio 2</label>
    <input type="radio" />
  </nve-radio>

  <nve-radio>
    <label>radio 3</label>
    <input type="radio" />
  </nve-radio>
  <nve-control-message>message</nve-control-message>
</nve-radio-group>
```
